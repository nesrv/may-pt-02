## Декораторы

### Задача 1

Декоратор печатает строку по шаблону и (как правило) пробрасывает результат дальше — чтобы не ломать композицию.

```python
def func_show(func):
    def wrapper(width, height, *args, **kwargs):
        value = func(width, height, *args, **kwargs)
        print(f"Площадь прямоугольника: {value}")
        return value

    return wrapper


@func_show
def get_sq(width, height):
    return width * height


get_sq(8, 11)
```

### Задача 2

В условии в примере номерации опечатка (`Пункт_1` дважды); на выходе ожидается нумерация `1..N` по фактическим словам.

```python
def show_menu(func):
    def wrapper(s: str):
        items = func(s)
        for i, name in enumerate(items, start=1):
            print(f"{i}. {name}")
        return items

    return wrapper


@show_menu
def get_menu(s: str):
    return s.split()


get_menu(input())
```

### Задача 4

Если в тексте есть «плохое» слово, возвращаем предупреждение без шаблона `текст … проверен`. Набор слов при необходимости расширяется.

```python
FORBIDDEN = ("бяка",)


def decorator(func):
    def wrapper(text: str):
        low = text.lower()
        for word in FORBIDDEN:
            if word in low:
                return f"Не пишите {word}!"
        return func(text)

    return wrapper


@decorator
def censor(text: str) -> str:
    return f"текст {text} проверен"


print(censor("Hello"))
print(censor("Какая-то бяка"))
```

### Задача 5 (логгер)

Одна строка лога — один JSON-объект (удобно потом парить построчно). Время в стиле примера из условия — через `ctime()`.

```python
import json
from datetime import datetime


def logger(func):
    def wrapper(*args, **kwargs):
        result = func(*args, **kwargs)
        record = {
            "Имя-функции": func.__name__,
            "Произвольные аргументы": list(args),
            "Именованные аргументы": kwargs,
            "Возвращаемое значение": result,
            "Время и дата вызова функции": datetime.now().ctime(),
        }
        with open("function_log.txt", "a", encoding="utf-8") as fp:
            fp.write(json.dumps(record, ensure_ascii=False) + "\n")
        return result

    return wrapper


@logger
def add_numbers(a, b):
    return a + b


add_numbers(2, 3)
```

def fib_seq(max_len: int):
    a, b = 1, 1
    for _ in range(max_len):
        yield a
        a, b = b, a + b


N = int(input())

### Задача 3 (пароли)

Бесконечный генератор: каждый `next` — новый пароль фиксированной длины.

```python
import random
from string import ascii_lowercase, ascii_uppercase

chars = ascii_lowercase + ascii_uppercase + "0123456789!?@#$*"


def passwords(n: int):
    while True:
        yield "".join(random.choice(chars) for _ in range(n))


N = int(input())
g = passwords(N)
for _ in range(5):
    print(next(g))
```

### Задача 3 (email `@mail.ru`)

Отдельная постановка в исходнике дублирует номер «3» — здесь отдельный генератор.

```python
import random
from string import ascii_lowercase, ascii_uppercase

letters = ascii_lowercase + ascii_uppercase


def random_mails(local_len: int):
    while True:
        local = "".join(random.choice(letters) for _ in range(local_len))
        yield f"{local}@mail.ru"


N = int(input())
gen = random_mails(N)
for _ in range(5):
    print(next(gen))
```

### Задача 3 (теорема Вильсона) — рабочий вариант проверки

Наивная формула `(n-1)! + 1` модуло `n` корректна для целых `n ≥ 2` по теореме Вильсона; для `n = 1` её обычно не используют — вынесите крайний случай отдельно. Для больших `n` факториал непрактичен; в учебных целях ниже — только малые простые.

```python
import math


def prime(n: int) -> None:
    if n < 2:
        print("Это составное число")
        return
    if (math.factorial(n - 1) + 1) % n != 0:
        print("Это составное число")
    else:
        print("Это простое число")
```

**Мини-комментарий:** тест Ферма `a**(n-1) % n == 1` даёт ложные простые (Carmichael) для некоторых составных `n`; для демонстрации используйте `pow(a, n - 1, n)` (быстро и без переполнения).

```python
def primFerma(a: int, n: int) -> None:
    if n < 2:
        print("составное")
        return
    if pow(a, n - 1, n) == 1:
        print("правдоподобно простое")
    else:
        print("составное")
```

### Задача 4 (`infinite`)

Циклический обход; пустой список — пустая строка; элементы сращиваются как в примере (`str` от каждого числа).

```python
def infinite(lst: list[int], tries: int) -> str:
    if not lst:
        return ""
    n = len(lst)
    return "".join(str(lst[i % n]) for i in range(tries))


print(infinite([2, 5, 8], 7))
print(infinite([], 1000))
print(infinite([7], 4))
```
