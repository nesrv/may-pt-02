## Самостоятельная работа (M2)

**Цель блока:** закрепить два паттерна — **декоратор как обёртка над callable** и **бесконечный генератор** как ленивый источник данных без заранее собранного списка.

**Формально:** декоратор реализует композицию `f' = d(f)`; генератор с `while True: yield ...` задаёт **корутину-потребителя** в смысле итерации (не путать с `asyncio`).

**По-бытовому:** декоратор — «хук до/после вызова», генератор — «фабрика бесконечных сэмплов»; главное не забыть, что `next()` двигает состояние, а не пересоздаёт вселенную.

### Задача 1 (меню + декоратор)

На вход поступает одна строка: названия пунктов меню через **пробел**.

Реализуйте функцию `get_menu(s)`, которая превращает строку в **список слов** (токенов) и возвращает его.

Определите декоратор `show_menu`, который **печатает** список в нумерованном виде и при этом не ломает обычный контракт вызова (возвращаемое значение — тот же список, если иное не оговорено в вашем варианте курса).

Формат вывода (подчёркнуто: в условиях часто опечатывают вторую строку — должны быть **разные** пункты):

```text
1. Пункт_1
2. Пункт_2
...
N. Пункт_N
```

Примените декоратор через `@show_menu`.

```python
def show_menu(func):
    ...


@show_menu
def get_menu(s):
    ...
```

### Задача 2 (список чисел + сортировка декоратором)

Вход — строка целых через пробел.

Функция `get_list(...)` должна вернуть список `int`. Декоратор (имя в шаблоне — `list_sort`) **сортирует по возрастанию** результат базовой функции; именно **отсортированный** список должен оказаться результатом вызова декорированной функции.

Вывод:

```python
lst = get_list(input())
print(*lst)
```

Пример:

- **Ввод:** `8 11 -5 4 3 10`  
- **Вывод:** `-5 3 4 8 10 11`

```python
def list_sort(func):

    return wrapper

lst = get_list(input())
print(*lst)
```

### Задача 3 (бесконечные `email`)

Задайте **функцию-генератор**, которая бесконечно выдаёт случайные адреса вида `<локальная_часть>@mail.ru`. Локальная часть длины `N` строится из символов `ascii_lowercase` и `ascii_uppercase` (импорт из `string`).

Пример при `N = 6`: `SCrUZo@mail.ru`.

Считайте `N` с клавиатуры. Напечатайте **ровно пять** первых адресов, каждый с новой строки.

> **Ввод:** `8`  
> **Вывод (пример структуры, значения случайны):**

```text
ESZYCIDP@mail.ru
YOPUMZGD@mail.ru
PAMNTYYA@mail.ru
WOIXZHSD@mail.ru
KAAAURAM@mail.ru
```

### Чек-лист перед сдачей

1. Декораторы возвращают **новую** функцию; не перепутайте порядок `return` в `wrapper`.
2. Для задачи 2 убедитесь, что сортируется именно список чисел, а не строка целиком.
3. Для задачи 3 не складывайте все бесконечные адреса в список — иначе получите классический **бесконечный цикл по памяти/времени**; достаточно пяти `next(...)` или цикла `for _ in range(5)`.
