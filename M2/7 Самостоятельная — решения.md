## Задача 1

```python
def show_menu(func):
    def wrapper(s: str):
        items = func(s)
        for i, title in enumerate(items, start=1):
            print(f"{i}. {title}")
        return items

    return wrapper


@show_menu
def get_menu(s: str):
    return s.split()


get_menu(input())
```

## Задача 2

```python
def list_sort(func):
    def wrapper(*args, **kwargs):
        data = func(*args, **kwargs)
        return sorted(data)

    return wrapper


@list_sort
def get_list(lst: str):
    return list(map(int, lst.split()))


lst = get_list(input())
print(*lst)
```

Параметр в шаблоне назван `lst`, по факту это строка ввода — тип в аннотации выше отражает реальное использование.

## Задача 3

```python
import random
from string import ascii_lowercase, ascii_uppercase

letters = ascii_lowercase + ascii_uppercase


def random_mails(local_len: int):
    while True:
        local = "".join(random.choice(letters) for _ in range(local_len))
        yield f"{local}@mail.ru"


N = int(input())
gen = random_mails(N)
for _ in range(5):
    print(next(gen))
```

При каждом запуске строки будут другими; важны домен `mail.ru`, длина локальной части `N` и алфавит.
