__Задача 1.__

Дан словарь, где ключ — имя студента, а значение — список его оценок за семестр.

```python
students = {
    "Анна": [5, 4, 4, 5],
    "Иван": [3, 4, 3, 4],
    "Мария": [5, 5, 5, 4],
    "Петр": [4, 4, 4, 4]
}
```

#### Требуется:

1. С помощью `map` вычислить **средний балл** каждого студента.
2. На основе полученных средних значений:

   * создать новый словарь вида

     ```python
     {"Анна": 4.5, "Иван": 3.5, ...}
     ```
3. Отсортировать словарь:

   * по среднему баллу **по убыванию**
4. Вывести результат в виде списка кортежей:

   ```python
   [("Мария", 4.75), ("Анна", 4.5), ("Петр", 4.0), ("Иван", 3.5)]
   ```

__Задача 2__ 

Реализуй функцию `analyze_orders`, которая обрабатывает произвольное количество заказов и дополнительных параметров.

```python
def analyze_orders(*args, **kwargs):
    ...
```

#### Входные данные

* `*args` — произвольное количество заказов.
  Каждый заказ — это кортеж вида:

  ```python
  (название_товара, цена, количество)
  ```

  Пример:

  ```python
  ("apple", 50, 3)
  ```

* `**kwargs` — дополнительные параметры обработки:

  * `discount` — процент скидки (целое число, по умолчанию 0)
  * `tax` — процент налога (целое число, по умолчанию 0)
  * `currency` — строка для отображения валюты (по умолчанию `"₽"`)

---

### 🎯 Требуется

1. Посчитать **общую стоимость** всех заказов.
2. Применить скидку (`discount`) и налог (`tax`) **в указанном порядке**:

   * сначала скидка
   * затем налог
3. Вернуть словарь с результатами:

   ```python
   {
       "items": количество_разных_товаров,
       "subtotal": сумма_без_скидок,
       "total": итоговая_сумма,
       "currency": валюта
   }
   ```

---

### 🧪 Пример вызова

```python
result = analyze_orders(
    ("apple", 50, 3),
    ("banana", 30, 2),
    discount=10,
    tax=5,
    currency="₽"
)
```

### ✅ Ожидаемый результат

```python
{
    "items": 2,
    "subtotal": 210,
    "total": 198.45,
    "currency": "₽"
}
```

__Задача 3__ 

Дан список сотрудников.
Каждый сотрудник представлен в виде кортежа:

```python
(name, department, salary, experience)
```

```python
employees = [
    ("Анна", "IT", 120000, 3),
    ("Иван", "HR", 80000, 5),
    ("Мария", "IT", 120000, 5),
    ("Петр", "IT", 100000, 3),
    ("Ольга", "HR", 80000, 2),
    ("Сергей", "IT", 100000, 5)
]
```

---

### 🎯 Требуется

Отсортировать список сотрудников **с помощью `sorted()` и `lambda`** по следующим правилам:

1. **По отделу** — в алфавитном порядке
2. **По зарплате** — по убыванию
3. **По стажу** — по убыванию
4. **По имени** — в алфавитном порядке (как дополнительный критерий)

Результат должен быть новым списком, исходный список менять нельзя.

---

### ✅ Ожидаемый формат результата

```python
[
    ("Ольга", "HR", 80000, 2),
    ("Иван", "HR", 80000, 5),
    ("Мария", "IT", 120000, 5),
    ("Анна", "IT", 120000, 3),
    ("Сергей", "IT", 100000, 5),
    ("Петр", "IT", 100000, 3)
]
```
