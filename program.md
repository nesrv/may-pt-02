# Курс «Продвинутый Python»

## Python для практических задач: анализ данных, backend, автоматизация и веб-визуализация

---

## Цель курса

Подготовить специалиста, способного использовать Python как универсальный инструмент для:

* анализа и обработки данных;
* автоматизации и работы с файлами;
* разработки backend-приложений;
* создания API и веб-визуализаций;
* реализации законченных программных продуктов.

---

## Портрет выпускника

Уровень: Junior+ / Middle Python Developer (Backend / Data-ориентированный)

На выходе выпускник умеет:

* писать чистый и поддерживаемый Python-код;
* работать с NumPy, Pandas, Matplotlib, SciPy;
* проектировать БД и работать через ORM;
* создавать REST API на FastAPI;
* извлекать данные из веб-источников;
* собирать backend-приложения «под ключ».

## Структура курса

## Модуль 1. Функциональное программирование

* Функция как объект. Паттерн «Фабрика функций»

Практические задания:

* Встроенные декораторы
